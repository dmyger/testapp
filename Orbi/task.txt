Тестовое задание ORBI

Необходимо написать программу для конвертации панорамных фотографий из эквиректангулярной проекции в кубическую.
На входе: файл изображения с эквиректангулярной проекцией некоторого панорамного изображения
(например https://upload.wikimedia.org/wikipedia/commons/8/83/Equirectangular_projection_SW.jpg).
На выходе: файл этого же изображения в формате проекции на 6 граней куба (порядок размещения граней куба
на выходном изображении может быть одним из приведенных на этом изображении:
https://docs.unity3d.com/560/Documentation/uploads/Textures/CubeLayout6Faces.png,
оси координат: http://postprocessor.su/pmpost/chpu2/rotary_axes.jpg).
Приложение должно собираться с помощью QtCreator под Qt5.9 без использования иных сторонних библиотек. 
GUI программы должен иметь простейшие элементы для импорта и экспорта файлов. А так же (не обязательно) область просмотра входного файла.
Крайне желательно реализовать GUI на QML.
Крайне желательно, что-бы непосредственно конвертация производилась на GPU с помощью шейдерной программы.
Исходный код приложения необходимо загрузить в открытый git-репозиторий (например github).
